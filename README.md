# 🎨 ArtCanvas – Digital Drawing Application

ArtCanvas is a feature-rich digital drawing application built using HTML5 Canvas and JavaScript. It includes advanced features like Undo/Redo functionality implemented using the Command Pattern, multiple drawing tools, custom backgrounds, file export options, and mobile touch support.

## 🚀 Live Demo
https://sherinebenezas.github.io/ArtCanvas/

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
- Canvas API
- LocalStorage

## ✨ Features
- Pencil, Eraser, and Shape tools
- Undo/Redo functionality
- Clear Canvas option
- Save drawing as image
- Responsive design
- Mobile touch support
